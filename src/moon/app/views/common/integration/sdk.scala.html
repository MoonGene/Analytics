Eclipse IDE - Installing the library

Step 1: Download the latest MoonGene Analytics library from here: http://www.moongene.com/sdk/android

Step 2: Import the library into your project's workspace. Go to File->Import->Android->Existing Android Code into Workspace. Browse for the folder with the library and then click Finish.

Step 3: Set library as a dwepedency of your projects. Go to your project properties, to the "Android" tab and then add the analytics library in the library section.

Step 4: Modify your AndroidManifest.xml permissions. We need to ensure that certain permissions are added.

        <!-- Internet is required to send events to the MoonGene Analytics -->
        <uses-permission android:name="android.permission.INTERNET" />

        <!-- This permissions is optional, but recommended to get more details on user's network
        and optimize batching of events -->
        <uses-permission android:name="android.permission.ACCESS_NETWORK_STATE" />

At this point, you're ready to use the MoonGene Analytics library inside Eclipse. Make sure that you import the library where you need it:

        import com.moongene.android.Analytics;
